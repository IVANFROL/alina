# Калькулятор таможенных платежей СИРИУС ОПТ

Веб-приложение для расчета таможенных платежей и формирования коммерческих предложений (КП) для импорта товаров из Китая.

## Функционал

- Загрузка справочника ТН ВЭД (Excel или JSON)
- Расчет таможенных платежей (пошлина, НДС, акциз)
- Расчет стоимости доставки (авто, ЖД, авиа, море)
- Формирование коммерческих предложений в формате Excel
- Автоматическая загрузка курсов валют с ЦБ РФ

## Использование

1. Откройте `index.html` в браузере
2. Загрузите файл справочника ТН ВЭД
3. Введите параметры товара
4. Нажмите "Рассчитать" или "Добавить товар"
5. Сформируйте КП в формате Excel

## Технологии

- HTML5, CSS3, JavaScript
- ExcelJS для генерации Excel файлов
- XLSX для чтения Excel файлов
