Настройки тарифов доставки

По умолчанию 50 юаней/кг. Если меняете валюту сделки, подставьте тариф в этой валюте.
Для авто/ЖД/моря доставка считается пропорционально объёму (м³) к полному контейнеру/фуре. Для авиа — по весу (кг).
